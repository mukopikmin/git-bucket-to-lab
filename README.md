# Migration app from GitBucket to GitLab

Migrate repositories on GItBucket to projects on GitLab.

## Working environment

- GitBucket: 4.33.0
- GitLab: 12.10.6

## Features

- [x] Repository (Including code, commits, tags, branches)
- [ ] Release
- [ ] Issue
- [ ] Pull request
- [ ] Wiki

## Install

Build client application.

    npm install
    npm run build

Start server.

    go run main.go
